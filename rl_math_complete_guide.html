Markov Property

The future depends only on the present тАФ not the history. This is what makes the math tractable.

P(sтВЬтВКтВБ | sтВЬ, aтВЬ, sтВЬтВЛтВБ, aтВЬтВЛтВБ, тАж)
  = P(sтВЬтВКтВБ | sтВЬ, aтВЬ)

// "The present state captures all relevant history"

Transition & Reward

P(sтА▓ | s, a)         transition probability

Expected reward:
r(s,a) = ╬г_{sтА▓} P(sтА▓|s,a) ┬╖ R(s,a,sтА▓)

Discounted Return

The quantity we want to maximize тАФ a geometric sum of future rewards.

GтВЬ = RтВЬтВКтВБ + ╬│┬╖RтВЬтВКтВВ + ╬│┬▓┬╖RтВЬтВКтВГ + тАж

// ╬│ < 1 ensures finite; ╬│тЖТ0 greedy; ╬│тЖТ1 far-sighted

Bellman
Equations

The central insight: optimal value functions satisfy a recursive consistency condition. This single idea powers all of RL.

State-Value V^╧А

V^╧А(s) = ЁЭФ╝_╧А[ GтВЬ | SтВЬ = s ]
       = ЁЭФ╝_╧А[ RтВЬтВКтВБ + ╬│┬╖GтВЬтВКтВБ | SтВЬ = s ]

V^╧А(s) = ╬г_a ╧А(a|s) ┬╖ ╬г_{sтА▓} P(sтА▓|s,a) [r(s,a) + ╬│┬╖V^╧А(sтА▓)]

Action-Value Q^╧А

Q^╧А(s,a) = ЁЭФ╝_╧А[ GтВЬ | SтВЬ=s, AтВЬ=a ]

Q^╧А(s,a) = ╬г_{sтА▓} P(sтА▓|s,a) [r(s,a) + ╬│ ╬г_{aтА▓} ╧А(aтА▓|sтА▓) Q^╧А(sтА▓,aтА▓)]

Relationship:
  V^╧А(s) = ╬г_a ╧А(a|s) ┬╖ Q^╧А(s,a)
  Q^╧А(s,a) = ╬г_{sтА▓} P(sтА▓|s,a) [r + ╬│┬╖V^╧А(sтА▓)]

Optimality Equations

V*(s)    = max_a  ╬г_{sтА▓} P(sтА▓|s,a) [r(s,a) + ╬│┬╖V*(sтА▓)]

Q*(s,a)  = ╬г_{sтА▓} P(sтА▓|s,a) [r(s,a) + ╬│ ┬╖ max_{aтА▓} Q*(sтА▓,aтА▓)]

Optimal policy:
  ╧А*(s) = argmax_a Q*(s,a)

Bellman Operator & Contraction

(ЁЭТп^╧А V)(s) = ╬г_a ╧А(a|s) ╬г_{sтА▓} P(sтА▓|s,a) [r + ╬│V(sтА▓)]
(ЁЭТп* V)(s)  = max_a ╬г_{sтА▓} P(sтА▓|s,a) [r + ╬│V(sтА▓)]

╬│-contraction:  тАЦЁЭТп*V тИТ ЁЭТп*UтАЦ_тИЮ тЙд ╬│ тАЦV тИТ UтАЦ_тИЮ

// Banach fixed-point theorem тЖТ unique V*, guaranteed convergence
тЪб
The contraction property (╬│ < 1) is the bedrock guarantee. Every RL convergence proof ultimately reduces to this.

Planning with
Dynamic Programming

When the model P(sтА▓|s,a) is known, we can plan exactly. The algorithms here are the conceptual ancestors of every modern RL method.

Policy Evaluation

Initialize VтВА arbitrarily;  V(terminal) = 0

For k = 0, 1, 2, тАж
  VтВЦтВКтВБ(s) тЖР ╬г_a ╧А(a|s) ╬г_{sтА▓} P(sтА▓|s,a) [r + ╬│┬╖VтВЦ(sтА▓)]

// VтВЦ тЖТ V^╧А as k тЖТ тИЮ  (by contraction, rate ╬│ per sweep)

Then: V^{╧АтА▓}(s) тЙе V^╧А(s)  for all s тИИ S

  V^╧А(s) тЙд Q^╧А(s, ╧АтА▓(s))
         = ╬г_{sтА▓} P(sтА▓|s,╧АтА▓) [r + ╬│┬╖V^╧А(sтА▓)]

Merge E + I into a single operator. Faster in practice.

Temporal Difference
Learning

No model. No full rollouts. Learn from a single step of experience by bootstrapping off your own estimates.

MC target
GтВЬ (full return)
TD(0) target
RтВЬтВКтВБ + ╬│V(SтВЬтВКтВБ)
Bias
TD: higher ┬╖ MC: zero

TD: lower ┬╖ MC: high

TD(0) Update

╬┤тВЬ = RтВЬтВКтВБ + ╬│┬╖V(SтВЬтВКтВБ) тИТ V(SтВЬ)   тЖР TD error

V(SтВЬ) тЖР V(SтВЬ) + ╬▒ ┬╖ ╬┤тВЬ

// Converges to V^╧А if ╬▒ satisfies Robbins-Monro:
//   ╬г ╬▒тВЩ = тИЮ,   ╬г ╬▒тВЩ┬▓ < тИЮ

n-step Returns

GтВЬ^(n) = RтВЬтВКтВБ + ╬│RтВЬтВКтВВ + тАж + ╬│тБ┐тБ╗┬╣RтВЬтВКтВЩ + ╬│тБ┐V(SтВЬтВКтВЩ)

n = 1:  TD(0)         (bootstrap after 1 step)
n = тИЮ:  Monte Carlo   (no bootstrap, full return)
n тИИ (1,тИЮ): interpolates bias / variance tradeoff

TD(╬╗) тАФ Eligibility Traces

╬╗-return (forward view):
  GтВЬ^╬╗ = (1тИТ╬╗) ╬г_{n=1}^тИЮ  ╬╗тБ┐тБ╗┬╣ GтВЬ^(n)

Eligibility trace (backward view):
  eтВЬ(s) = ╬│╬╗ ┬╖ eтВЬтВЛтВБ(s) + ЁЭЯЩ[SтВЬ = s]

Update all states simultaneously:
  V(s) тЖР V(s) + ╬▒ ┬╖ ╬┤тВЬ ┬╖ eтВЬ(s)

// ╬╗=0: TD(0);  ╬╗=1: offline MC;  ╬╗тИИ(0,1): best of both

Q-Learning &
SARSA

Apply TD ideas to action-values. SARSA follows the current policy; Q-learning learns the optimal policy regardless of behavior.

SARSA тАФ On-Policy

Tuple: (SтВЬ, AтВЬ, RтВЬтВКтВБ, SтВЬтВКтВБ, AтВЬтВКтВБ)   тЖР action from ╧А

Q(SтВЬ,AтВЬ) тЖР Q(SтВЬ,AтВЬ) + ╬▒[RтВЬтВКтВБ + ╬│┬╖Q(SтВЬтВКтВБ,AтВЬтВКтВБ) тИТ Q(SтВЬ,AтВЬ)]

Q-Learning тАФ Off-Policy

Tuple: (SтВЬ, AтВЬ, RтВЬтВКтВБ, SтВЬтВКтВБ)

Q(SтВЬ,AтВЬ) тЖР Q(SтВЬ,AтВЬ) + ╬▒[RтВЬтВКтВБ + ╬│┬╖max_{aтА▓} Q(SтВЬтВКтВБ,aтА▓) тИТ Q(SтВЬ,AтВЬ)]

// Behavior policy b can be anything (╬╡-greedy, randomтАж)
// Still converges to Q*

Convergence Theorem (Watkins & Dayan 1992)

Q-learning тЖТ Q* with probability 1 iff:
  1. All (s,a) pairs visited infinitely often
  2. ╬г_t ╬▒тВЬ(s,a) = тИЮ   and   ╬г_t ╬▒тВЬ┬▓(s,a) < тИЮ
  3. Rewards bounded:  |R| тЙд R_max

╬╡-Greedy Exploration

╧А_╬╡(a|s) = тОз 1 тИТ ╬╡ + ╬╡/|A|   if a = argmax Q(s,┬╖)
           тОй ╬╡ / |A|           otherwise

╬╡ тЖТ 0:  pure exploitation  |  ╬╡ = 1:  pure exploration

Double Q-Learning (van Hasselt 2010)

Two networks Q_A, Q_B:

Q_A(S,A) тЖР Q_A + ╬▒[R + ╬│┬╖Q_B(SтА▓, argmax_{aтА▓} Q_A(SтА▓,aтА▓)) тИТ Q_A(S,A)]

// Decouples selection from evaluation тЖТ kills overestimation bias
1995тАУ2012 ┬╖ Scaling Up

Function
Approximation

Tabular methods collapse for Atari, robotics, language. We parameterize value functions with linear models and neural networks.

Linear Approximation

V╠В(s; w) = wс╡А ╧Ж(s)     ╧Ж(s) тИИ тДЭтБ┐  (feature vector)

  = w + ╬▒ ┬╖ ╬┤тВЬ ┬╖ ╧Ж(SтВЬ)

  w тЖР w + ╬▒ [R + ╬│V╠В(SтА▓;w) тИТ V╠В(S;w)] ┬╖ ╧Ж(S)

Neural Network Approximation

TD target:
  y = R + ╬│ ┬╖ max_{aтА▓} Q(sтА▓,aтА▓; ╬╕)

Loss:
  L(╬╕) = ЁЭФ╝[ (y тИТ Q(s,a;╬╕))┬▓ ]

  тИЗ_╬╕ L = тИТ2(y тИТ Q(s,a;╬╕)) ┬╖ тИЗ_╬╕ Q(s,a;╬╕)
тЪа
Deadly Triad (Tsitsiklis & Van Roy 1997): combining (1) off-policy + (2) function approximation + (3) bootstrapping can diverge. DQN fixes this with experience replay + target networks.

  v тЖР v + ╬▓(╬┤тВЬ тИТ ╧ЖтВЬс╡АvтВЬ) ╧ЖтВЬ    ╬▓ тЙл ╬▒

// Converges with all three deadly-triad elements active

Policy Gradient
Methods

Skip value functions entirely. Directly differentiate through the policy and climb the expected-return surface.

Objective

J(╬╕) = V^{╧А_╬╕}(sтВА) = ЁЭФ╝_{╧Д~╧А_╬╕}[GтВА]
     = ЁЭФ╝_{╧Д~╧А_╬╕}[ ╬гтВЬ ╬│с╡Ч RтВЬтВКтВБ ]

╧Д = (sтВА, aтВА, rтВБ, sтВБ, тАж)  тАФ trajectory

Policy Gradient Theorem (Sutton 1999)

тИЗ_╬╕ J(╬╕) = ЁЭФ╝_{s~d^╧А, a~╧А_╬╕} [ Q^╧А(s,a) ┬╖ тИЗ_╬╕ log ╧А_╬╕(a|s) ]

d^╧А(s) = (1тИТ╬│) ╬г_{t=0}^тИЮ  ╬│с╡Ч P(SтВЬ=s|╧А)   (discounted state dist.)

// No need to differentiate through environment dynamics!

Log-Derivative Trick

тИЗ_╬╕ ЁЭФ╝_{x~p_╬╕}[f(x)] = ЁЭФ╝_{x~p_╬╕}[ f(x) ┬╖ тИЗ_╬╕ log p_╬╕(x) ]

Derivation:
  тИЗ_╬╕ тИл p_╬╕(x) f(x) dx
  = тИл тИЗ_╬╕ p_╬╕(x) ┬╖ f(x) dx
  = тИл p_╬╕(x) ┬╖ тИЗ_╬╕ log p_╬╕(x) ┬╖ f(x) dx  тЬУ

// Also called: score function estimator, REINFORCE trick

REINFORCE (Williams 1992)

Sample ╧Д ~ ╧А_╬╕;  compute returns GтВЬ тИАt

╬╕ тЖР ╬╕ + ╬▒ ┬╖ ╬гтВЬ GтВЬ ┬╖ тИЗ_╬╕ log ╧А_╬╕(AтВЬ|SтВЬ)

With baseline b(s) (e.g. V(s)):
  ╬╕ тЖР ╬╕ + ╬▒ ┬╖ ╬гтВЬ (GтВЬ тИТ b(SтВЬ)) ┬╖ тИЗ_╬╕ log ╧А_╬╕(AтВЬ|SтВЬ)

// Baseline reduces variance; gradient stays unbiased
//   ЁЭФ╝[ b(S) ┬╖ тИЗ_╬╕ log ╧А_╬╕ ] = 0  (proof via log-deriv trick)

Natural Policy Gradient

Fisher Information Matrix:
  F_╬╕ = ЁЭФ╝[ тИЗ_╬╕ log ╧А_╬╕ ┬╖ тИЗ_╬╕ log ╧А_╬╕с╡А ]

Standard gradient:  ╬╕ тЖР ╬╕ + ╬▒ тИЗ_╬╕ J
Natural gradient:   ╬╕ тЖР ╬╕ + ╬▒ F_╬╕тБ╗┬╣ тИЗ_╬╕ J

// Steepest ascent in KL geometry, not Euclidean parameter space
// Invariant to reparameterization of ╧А_╬╕
2000тАУ2015 ┬╖ Actor-Critic

Actor-Critic
& GAE

Use a critic (value function) to reduce variance of the actor's gradient estimate. The architecture behind nearly every modern algorithm.

Advantage Function

A^╧А(s,a) = Q^╧А(s,a) тИТ V^╧А(s)

  A > 0:  action a is better than average under ╧А
  A < 0:  action a is worse than average under ╧А
  A = 0:  indifferent

One-step TD estimate:
  ├ВтВЬ тЙИ ╬┤тВЬ = RтВЬтВКтВБ + ╬│┬╖V(SтВЬтВКтВБ) тИТ V(SтВЬ)

Actor-Critic Updates

// Actor (policy) тАФ climbs advantage-weighted log-prob:
╬╕ тЖР ╬╕ + ╬▒_╬╕ ┬╖ ├ВтВЬ ┬╖ тИЗ_╬╕ log ╧А_╬╕(AтВЬ|SтВЬ)

// Critic (value) тАФ minimizes TD error:
w тЖР w + ╬▒_w ┬╖ ╬┤тВЬ ┬╖ тИЗ_w V_w(SтВЬ)

GAE тАФ Generalized Advantage Estimation (Schulman 2015)

╬┤тВЬ^V = RтВЬ + ╬│┬╖V(SтВЬтВКтВБ) тИТ V(SтВЬ)

├ВтВЬ^{GAE(╬│,╬╗)} = ╬г_{l=0}^тИЮ (╬│╬╗)╦б ╬┤тВЬтВКтВЧ^V
             = ╬┤тВЬ + ╬│╬╗┬╖╬┤тВЬтВКтВБ + (╬│╬╗)┬▓┬╖╬┤тВЬтВКтВВ + тАж

╬╗=0:  ├ВтВЬ = ╬┤тВЬ                   (low var, high bias)
╬╗=1:  ├ВтВЬ = ╬г_{l} ╬│╦б ╬┤тВЬтВКтВЧ       (high var, low bias)
// ╬╗ тИИ (0,1): continuous bias-variance interpolation

TRPO тАФ Trust Region (Schulman 2015)

max_╬╕  ЁЭФ╝тВЬ[ ╧А_╬╕(aтВЬ|sтВЬ)/╧А_{╬╕_old}(aтВЬ|sтВЬ) ┬╖ ├ВтВЬ ]
subject to:  ЁЭФ╝тВЬ[ KL[╧А_{╬╕_old}(┬╖|s) тАЦ ╧А_╬╕(┬╖|s)] ] тЙд ╬┤

Monotonic improvement theorem:
  J(╧А_new) тЙе J(╧А_old) тИТ C ┬╖ max_s KL[╧А_old тАЦ ╧А_new]

// Guaranteed improvement every step. Expensive to solve.

SAC тАФ Soft Actor-Critic (Haarnoja 2018)

J(╧А) = ╬гтВЬ ЁЭФ╝[ RтВЬ + ╬▒ ┬╖ H(╧А(┬╖|sтВЬ)) ]

Soft Bellman:
  V^╧А(s)   = ЁЭФ╝_a[ Q^╧А(s,a) тИТ ╬▒ log ╧А(a|s) ]
  Q^╧А(s,a) = r + ╬│┬╖ЁЭФ╝_{sтА▓}[ V^╧А(sтА▓) ]

// ╬▒ balances reward vs. exploration entropy; auto-tunable
2013тАУ2020 ┬╖ Deep RL

Deep Reinforcement
Learning

Combine deep neural networks with RL. DQN beats humans at Atari. PPO becomes the standard policy optimizer for everything from robotics to LLMs.

DQN (Mnih et al. 2013/2015)

Two stabilizing tricks:

1. Experience Replay:  buffer D stores (s,a,r,sтА▓)
   тЖТ sample random minibatch тЖТ breaks temporal correlations

2. Target Network ╬╕тБ╗:  frozen copy, updated every C steps
   тЖТ y = r + ╬│ ┬╖ max_{aтА▓} Q(sтА▓,aтА▓; ╬╕тБ╗)

Loss:
  L(╬╕) = ЁЭФ╝_{(s,a,r,sтА▓)~D} [(y тИТ Q(s,a;╬╕))┬▓]

PPO тАФ Proximal Policy Optimization (Schulman 2017)

TRPO simplified: clip the importance ratio instead of solving a constrained optimization problem.

Ratio:  rтВЬ(╬╕) = ╧А_╬╕(aтВЬ|sтВЬ) / ╧А_{╬╕_old}(aтВЬ|sтВЬ)

L^{CLIP}(╬╕) = ЁЭФ╝тВЬ [ min(
    rтВЬ(╬╕) ┬╖ ├ВтВЬ,
    clip(rтВЬ(╬╕), 1тИТ╬╡, 1+╬╡) ┬╖ ├ВтВЬ
) ]

Full PPO loss:
  L(╬╕) = тИТL^{CLIP} + cтВБ┬╖L^{VF} тИТ cтВВ┬╖S[╧А_╬╕]

  L^{VF} = (V_╬╕(sтВЬ) тИТ VтВЬ^{targ})┬▓
  S      = H(╧А_╬╕(┬╖|sтВЬ))          тЖР entropy bonus
  cтВБтЙИ0.5,  cтВВтЙИ0.01,  ╬╡тЙИ0.2

PPO Training Loop

1
Collect T timesteps with ╧А_{╬╕_old} тЖТ fill rollout buffer
2
Compute advantages via GAE(╬│, ╬╗)
3
For K epochs, minibatch SGD:╬╕ тЖР ╬╕ + ╬▒ ┬╖ тИЗ_╬╕ L^{CLIP+VF+S}(╬╕)
4
Update ╬╕_old тЖР ╬╕, go to 1

TD3 тАФ Twin Delayed DDPG (Fujimoto 2018)

Twin critics: Q_{╬╕тВБ}, Q_{╬╕тВВ}

Target (min of twins):
  y = r + ╬│ ┬╖ min(Q_{╬╕тВБтБ╗}, Q_{╬╕тВВтБ╗})(sтА▓, ├г)
  ├г = ╧А_{╧ЖтБ╗}(sтА▓) + clip(╬╡, тИТc, c)   тЖР smoothed target

Policy update (delayed тАФ every 2 critic steps):
  тИЗ_╧Ж J = ЁЭФ╝_s[тИЗ_a Q_{╬╕тВБ}(s,a)|_{a=╧А_╧Ж(s)} ┬╖ тИЗ_╧Ж ╧А_╧Ж(s)]
